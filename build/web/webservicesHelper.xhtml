List all races

To get all races you need to send a GET request with the page you want:

/api/track/{page}?username=&mdp=your password

Post a new track

To post a new track you need to send a POST request with the track content inside:

/api/track?username=&mdp=your password

Body :
{
   "speed":9.0,
  "latitude":42.34,
  "longitude": 15.50
}

Delete a track :

To delete a track send a DELETE request on this url:

/api/track/{trackId}?username=&mdp=your password
